## Music
Original music was copyrighted, back-up music provided in `assets`. Simply change the `"filename"` attribute in this section of a level file:
```json
"sounds": [
        {
            "filename": "prologue.mp3",
            "loop_forever": true,
            "name": "background"
        }
    ],
```
